# Portafolio Profesional – Francisco Calvo Rodriguez

Este es un sitio web de portafolio moderno, altamente visual y profesional, diseñado para destacar proyectos de automatización, desarrollo de datos e inteligencia artificial. El objetivo principal es ofrecer una experiencia de usuario premium que demuestre habilidades técnicas y atención al detalle estético.

---

## 🚀 Tecnologías Utilizadas

| Tecnología | Propósito | ¿Por qué? |
| :--- | :--- | :--- |
| **HTML5** | Estructura semántica | Para asegurar la accesibilidad (A11y) y un buen posicionamiento SEO. |
| **Vanilla CSS** | Diseño y Estilo | Se utilizó un sistema de variables (tokens) para mantener la consistencia visual y facilitar cambios globales de diseño sin dependencias externas. |
| **Vanilla JavaScript** | Interactividad y Lógica | Para manejar efectos dinámicos, validación de formularios y animaciones sin la sobrecarga de un framework pesado. |
| **Lucide Icons** | Iconografía | Proporciona un conjunto de iconos minimalista y consistente que se integra perfectamente con el diseño. |
| **EmailJS** | Integración de Contacto | Permite el envío de correos directamente desde el cliente sin necesidad de configurar un servidor backend propio. |
| **Google Fonts** | Tipografía | Se seleccionaron *Inter* y *JetBrains Mono* para equilibrar profesionalismo con un toque moderno y técnico. |

---

## ✨ Características Principales

*   **Diseño Narrativo Premium:** Estética en modo oscuro con efectos de *glassmorphism* (vidrio esmerilado) y gradientes vibrantes.
*   **Fondos "Aurora":** Animaciones fluidas en el fondo que dan vida y dinamismo al sitio sin sacrificar rendimiento.
*   **Interactividad Completa:**
    *   **Cursor Personalizado:** Una experiencia de navegación única que responde a elementos interactivos.
    *   **Efecto Typewriter:** Dinamismo en la presentación del perfil profesional.
    *   **Scroll Reveal:** Animaciones suaves al navegar por las secciones para una lectura progresiva.
*   **Dark/Light Mode:** Soporte completo para temas claro y oscuro ajustado a la preferencia del usuario.
*   **Totalmente Responsivo:** Diseño adaptado para una experiencia fluida desde smartphones hasta pantallas de escritorio de gran formato.
*   **SEO Optimizado:** Inclusión de etiquetas Meta y OpenGraph para mejorar el compartido en redes sociales y la visibilidad en buscadores.

## 🛠️ Instalación y Uso

No requiere compilación. Simplemente clona el repositorio y abre el archivo `index.html` en tu navegador favorito o utiliza una extensión como *Live Server* en VS Code.

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/nombre-del-repo.git

# Entrar a la carpeta
cd nombre-del-repo

# Abrir index.html
```
